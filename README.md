# DiamondPattern
Program w C rysujący diament ze znaków

## Przykład wejścia (n = 15,znak = X)

## Jak uruchomić?
1. Sklonuj repozytorium: `git clone https://github.com/TwojUzytkownik/DiamondPattern.git`
2. Skompiluj: `gcc diamond.c -o diamond`
3. Uruchom: `./diamond`
4. Podaj szerokość i znak (np. 6 i #).

## Funkcjonalności
- Wczytuje szerokość i znak od użytkownika.
- Zapewnia parzystą szerokość diamentu.
- Używa jednej pętli z `abs` dla efektywności.
